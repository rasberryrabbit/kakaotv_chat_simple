* localhost:8090 : 기본 http 접속 URL로 doc폴더의 파일이 보여집니다. 
              doc/img 폴더에는 로컬에서 이미지 표현을 위해서 이모티콘 이미지가 저장됩니다.


* httplog.htm : 업데이트 되는 채팅의 데이터가 들어오는 URL. 
              GET 파라미터와 응답 헤더에서 ChatIndex로 값을 주고 받으면서 최근 업데이트된 채팅 내용만을 가져올 수 있습니다.
              GET 파라미터가 없는 경우에는 버퍼에 있는 전체 채팅의 내용을 받아옵니다.


* upload.action.htm : 파일을 업로드하는 POST 주소입니다. 
              Form의 주소로 사용하면 개별적인 업로드 환경을 만들 수 있습니다.

* httpalertlog.htm : 알림을 전달하는 데이터 페이지.


* websocket 방식을 지원. websocket방식은 오버헤드가 get방식보다 덜하고 브라우져가 지원되면 사용이 가능합니다.
  주기적으로 확인하는 GET방식보다 더 낫습니다.

  - 포트 8092 : 채팅 데이터가 전달.
  - 포트 8094 : 알림 데이터가 전달.
